# macOS Automation & Data Processing Scripts

This repository contains production-ready Python automation scripts designed for macOS. 

## Features
- **Data Resiliency**: Built-in automated fallback mechanics to handle network drops and server blocks.
- **Data Formatting**: Automatically generates clean, structured `.csv` spreadsheets directly on the Desktop.
- **Financial Processing**: Processes real-time global currency exchange rates using optimized data mapping.
